Fight Mechanics

This is a 2 Phase encounter, but I'm going to break up Phase 1 into the different orb mechanics you'll be dealing with for each set of Essences that spawn. When an Essence is absorbed there are 3 different things that happen per essence: An add will spawn, there will be an initial burst of damage or healing absorb applied, and players will be affected by that Essences' mechanic which will need to be dealt with.

Phase 1 - Kill Void, Absorb Nightmare and Vita: This is what most guilds opt to do for their 1st set of Essence spawns. When the boss absorbs Nightmare, players will need to be spread by 5 yards to avoid cleaving with Unleashed Nightmare, a Night Terror will spawn, and 5 stacks of Unstable Nightmare will affect a random player. The player with Unstable Nightmare will want to run close to melee so that melee can start rotating through the Unstable Nightmare mechanic.
Additionally, Vita will be absorbed causing Unleashed Vita to do initial heavy damage, a Crackling Stalker to spawn, and Unstable Vita to affect a random player. The player with Unstable Vita should go to your designated first marker to allow the next person in the ranged group to take the next hit. Any players not dealing with either Unstable Nightmare or Unstable Vita need to be killing the Crackling Stalker > Boss > Night Terror.

Phase 1 - Kill Vita, Absorb Nightmare and Void: This is what most guilds opt to do for their 2nd set of Essence spawns. The boss will again absorb Nightmare so the same mechanics will apply - the only change is that most groups opt to have their ranged deal with the Unstable Nightmare mechanic this time, as melee have a far easier time to deal with the Unstable Void mechanic.
When the boss absorbs Void it will trigger Unleashed Void, creating a big healing absorb shield, spawn a Void Hunter, and cause the first Unstable Void circle to spawn, which needs to be soaked by a player. It is worth noting that Unstable Void will travel in the direction you're facing when it hits, so you can face it into a wall or object to make it stay near the boss on the wall without any trouble. Anyone not dealing with Unstable Void or Nightmare need to be killing the Void Hunter > Boss > Night Terror. The Void Hunter casts Void Collapse which needs to either be soaked by a large group of players or need to be immuned - you should aim to only have 2 casts of this go off or else it will be much harder to deal with.

Phase 1 - Kill Nightmare, Absorb Vita and Void: This is what most guilds opt to do for their 3rd set of Essence Spawns. All of the mechanics at this point have been seen already, Ranged should deal with the Unstable Vita mechanic while melee deal with the Unstable Void mechanic. Anyone not dealing with these mechanics should be killing Void Hunter > Crackling Stalker > Boss.

If your group chooses to use Heroism/Bloodlust on pull, you should aim to phase the boss at 40% before he absorbs any of the orbs, otherwise you repeat the first set of Essences and then phase the boss after you're finished dealing with the mechanics.

Phase 2: Once the boss hits 40% all Essences despawn and Ruin starts ticking all players for heavy damage. Tanks will need to be sitting at roughly half health as much as possible so they don't die to Decaying Strike while healers make sure to not top off players who have Corrupted Existence. Occasionally there will be large purple circles on the floor, Void Eruption, that need to be dodged, as well as Charged bonds linking a few players together. Players simply need to run away to remove the bonds.

Suggested Talents

A standard talent build is recommended on this fight. All of the Tier 60 talents are viable on this fight, however most opt to run either Ancestral Protection Totem or Ancestral Vigor to help keep players healthy during one of the Essence combination overlaps in Phase 1.

Notable Azerite Traits and Essences

There is a lot of damage on this fight, however there are times where you're unable to be stacked up because of Unleashed Nightmare - this doesn't necessarily hurt traits like Spouting Spirits, just limits it's usefulness, however all of our standard traits like Overflowing Shores, Turn of the Tide, and even Ancestral Resonance are also great picks for this fight if you choose to not take multiple Spouting Spirits. For essences I would recommend taking Vision of Perfection major as there is a lot of damage happening on the fight and people are very rarely topped off.

Healing Tips

  •  Cooldowns like Healing Tide Totem are very good at healing off the Unleashed Void healing absorb.
  • Figure out where you want to place Ancestral Protection Totem if you choose to take it - it's most often used during one of the Essence overlaps that your group struggles with.
  •  If you're using the Forbidden Obsidian Claw trinket, you can use the first use on the boss on pull and the 2nd use lines up perfectly with the 3rd Essence set - this allows you to put in a good amount of damage on the Nightmare Essence.
  • Make sure you're tracking Corrupted Existence so you don't accidentally kill someone in Phase 2, but don't be scared to not heal at all - you're more likely to lose people to the initial application of Corrupted Existence than you are just healing someone to full.
